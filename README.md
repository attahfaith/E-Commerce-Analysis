# Sales Performances Analysis

 ### Project Overview

This project analyzes and compares retail sales performance between 2020 and 2021 across multiple retail chains. The analysis focuses on understanding sales trends, profit margins, product category performance and regional distribution to help the business identify growth opportunities and areas of decline.

### Data Source

Sales Performance: The dataset used for this analyis is the "E-Commerce_data.csv" file, containing a detailed information about the analysis of the company

### Tools

- Excel - Data cleaning, preparation and analysis
- Power Query (Excel)- Data transformation, removing duplicates, and filtering data
- Pivot Tables - Summarizing and aggregating sales data
- Excel Chart - Data Visualization and trend analysis


 ### Exploratory Data Analysis

EDA involves exploring the E-Commerce  data to answer key questions, such as;

- Which product category is most profitable?
- Which months shows the biggest revenue difference between 2020 and 2021?
- Which retailer generated the highest sales in 2020 vs 2021?

   ### Data Analysis

   Include some interesting codes\features worked with

- Pivot Tables - summarized sales by region, retailer and product category
- Charts - bar charts and line graphs for trend analysis
- Data Validation - ensure clean data entry
- IF & SUMIF Functions - calculated year totals

   ### Results\findings

  The sales analysis reveals that 2021 significantly outperformed 2020 across all retail chain. Revenue peaked mid-year between May and August by 2021 reaching up to $80k, suggesting a strong seasonal demand patterns.
